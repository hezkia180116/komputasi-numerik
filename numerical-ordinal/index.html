    Numerical Solutions of Ordinary Differential Equations¶

    Dalam bab ini kita membahas metode numerik untuk ODE . Kami akan membahas dua metode dasar, Metode Euler dan Runge-Kutta Metode.

    Metode Euler¶

    Metode Euler. Metode Euler adalah metode yang paling sederhana dalam menemukan solusi perkiraan untuk persamaan orde pertama. Dari depan rumus perbedaan

    f'(x) ≈ \frac{f(x + h) - f(x)}{h}

    kita punya $$ f(x + h) ≈ f(x) + f'(x)h $$

    Sekarang jika x'(t) = f(t, x) adalah persamaan diferensial orde satu, berlaku (1), kita punya

    x(t + h) ≈ x(t) + f(t, x(t))h.

    Misalkan kita ingin mencari solusi aproksimasi pada interval [a, b] dengan nilai awal x(a) = x0, interval dibagi menjadi n subinterval masing-masing dengan panjang h = b−a / n, ujung-ujung subintervalnya adalah t0 = a, t1 = a + h, t2 = a + 2h, · · · , tn = a + nh = b. Mulai dengan x0 kita dapat menghitung

    x_1 = x_0 + f(t_0, x_0)h
    x_2 = x_1 + f(t_1, x_1)h
    .
    x_n = x_n−1 + f(t_n−1, x_n−1)h

    Contoh¶

    Soal Temukan perkiraan untuk x(1) jika x't = t^2 - e^x sin(t), x(0) = 1 dengan h= 0,25.

    Solusi Di sini intervalnya adalah [0, 1], jadi a = 0, b = 1. Karena h = (\frac{b−a}{n}) = 0,25 kita memiliki n = 4 dan kita perlu menghitung x1, x2, x3, x4 mulai dengan x0 = x(0) = 1.

    x_1 = x_0 + f(t_0, x_0)h = 1 + f(0, 1) ∗ 0.25 = 0.15853
    x_2 = x_1 + f(t_1, x_1)h = 0.15853 + f(0.25, 0.15853) ∗ 0.25 = 0.01833
    x_3 = x_2 + f(t_2, x_2)h = 0.01833 + f(0.5, 0.01833) ∗ 0.25 = 0.23811
    x_4 = x_3 + f(t_3, x_3)h = 0.23811 + f(0.75, 0.15853) ∗ 0.25 = 0.30128

    Jadi pendekatan untuk x(1) adalah x(1) x4= 0,30128. Juga x(0,25) x1 = 0,15853, x(0,5) x2 = 0,01833, x(0,75) x3 = 0,23811.

    Runge-Kutta Method¶

    Ada beberapa metode Runge-Kutta, tetapi metode Runge-Kutta urutan keempat paling populer. Misalkan x'(t) = f(t, x), x(a) = x0. dan h = b−a / n adalah panjang langkah, Metode Runge-Kutta orde keempat diberikan di bawah ini,

    k_1 = hf(t_i, x_i)
    k_2 = hf(t_i + \frac{1}{2} h, x_i + \frac{1}{2} k_1)
    k_3 = hf(t_i + \frac{1}{2}h, x_i + \frac{1}{2}k_2)
    k_4 = hf(t_i + h, x_i + k_3)
    x_{i+1} = x_i + \frac{1}{6}(k_1 + 2k_2 + 2k_3 + k_4)

    Secara umum metode Runge-Kutta memberikan hasil yang lebih akurat dibandingkan dengan metode Euler pada panjang langkah yang sama. Namun, metode Runge-Kutta lebih mahal untuk diterapkan, yaitu, pada setiap langkah, Runge-Kutta membutuhkan lebih banyak komputasi daripada metode Euler (empat evaluasi fungsi dibandingkan satu dalam metode Euler).

    Contoh¶

    Soal Temukan perkiraan untuk x(1) jika x't = t^2 - e^x sin(t), x(0) = 1 dengan h= 0,25.

    Solusi Di sini intervalnya adalah [0, 1], jadi a = 0, b = 1. Karena h = \frac{b-a}{n} = 0,25 kita memiliki n = 4 dan kita perlu menghitung x1, x2, x3, x4 mulai dengan x0 = x(0) = 1.

    x_1:
    k_1 = hf(t_0, x_0) = 0.25 ∗ f(0, 1) = 0
    k_2 = hf(t_0 + \frac{1}{2}h, x_0 + \frac{1}{2}k_1) = f(0.125, 1) = −0.373276
    k_3 = hf(t_0 + \frac{1}{2}h, x_0 + \frac{1}{2}k_2) = 0.25f(0.125, 1 + 0.5(−0.373276)) = −0.373276
    k_4 = hf(t_0 + h, x_0 + k_3) = 0.25f(0.25, 1 − 0.373276) = −0.309854
    x_1 = x0 + \frac{1}{6} (k_1 + 2k_2 + 2k_3 + k_4)
    = 1 + \frac{1}{6} (0 + 2 ∗ (−0.373276) + 2 ∗ (−0.373276) − 0.309854)
    = 0.923911
    t_1 = t_0 + h = 0 + 0.25 = 0.25
    x_2:
    k_1 = hf(t_1, x_1) = 0.25 ∗ f(.25, 0.923911) = −0.560741
    k_2 = hf(t_1 + \frac{1}{2}h, x_1 + \frac{1}{2}k_1) = f(0.125, 0.923911 + 0.5(−0.560741)) = −0.719601
    k_3 = hf(t_1 + \frac{1}{2}h, x_1 + \frac{1}{2}k_2) = 0.25f(0.125, 0.923911 + 0.5(−0.719601)) = −0.702688
    k_4 = hf(t_1 + h, x_1 + k_3)
    = 0.25f(0.5, 0.923911 − 0.702688) = −0.763158
    x_2 = x_1 + \frac{1}{6}(k_1 + 2k_2 + 2k_3 + k_4)
    = 0.923911 + \frac{1}{6}(−0.560741 + 2 ∗ (−0.719601) + 2 ∗ (−0.702688) − 0.763158) = 0.750224
    t_2 = t_1 + h = 0.25 + 0.25 = 0.5
    x_3:
    k_1 = hf(t_2, x_2) = 0.25 ∗ f(.5, 0.750224) = −0.765171
    k_2 = hf(t_2 + \frac{1}{2}h, x_2 + \frac{1}{2}k_1)
    = f(0.625, 0.750224 + 0.5(−0.765171)) = −0.735295
    k_3 = hf(t_2 + \frac{1}{2}h, x_2 + \frac{1}{2}k_2)
    = 0.25f(0.625, 0.750224 + 0.5(−0.735295)) = −0.739508
    k_4 = hf(t_2 + h, x_2 + k_3)
    = 0.25f(0.75, 0.750224 − 0.739508) = −0.637224
    x_3 = x_2 + \frac{1}{6}(k_1 + 2k_2 + 2k_3 + k_4)
    = 0.750224 + \frac{1}{6} (−0.765171 + 2 ∗ (−0.735295) + 2 ∗ (−0.739508) − 0.637224)
    = 0.568891
    t_3 = t_2 + h = 0.5 + 0.25 = 0.75
    x_4:
    k_1 = hf(t_3, x_3) = 0.25 ∗ f(0.75, 0.568891) = −0.641483
    k_2 = hf(t_3 + \frac{1}{2}h, x_3 + \frac{1}{2}k_1)
    = f(0.875, 0.568891 + 0.5(−0.641483)) = −0.485628
    k_3 = hf(t_3 + \frac{1}{2}h, x_3 + \frac{1}{2}k_2)
    = 0.25f(0.875, 0.568891 + 0.5(−0.485628)) = −0.510243
    k_4 = hf(t_3 + h, x_3 + k_3)
    = 0.25f(1, 0.568891 − 0.510243) = −0.308297
    x_4 = x_3 + \frac{1}{6}(k_1 + 2k_2 + 2k_3 + k_4)
    = 0.568891 + \frac{1}{6} (−0.641483 + 2 ∗ (−0.485628) + 2 ∗ (−0.510243) − 0.308297)
    = 0.446327
    t_4 = t_3 + h = 0.75 + 0.25 = 1.0

    Jadi pendekatan untuk x(1) adalah x(1) ≈ x4 = 0,446327.
